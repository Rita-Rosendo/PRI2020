    • Abstract:

    Foi-nos proposta a realização de um trabalho de casa que tem como objectivo criar um servidor em NodeJs que recebe pedidos do tipo GET na porta 3001 e responde com uma lista com a informação requisitada ou então um índice.
    Quando se trata de um pedido http://localhost:3001/alunos, o servidor responde com uma lista dos alunos que pertencem à escola de música.
    Quando se trata de um pedido http://localhost:3001/cursos, o servidor responde com uma lista dos cursos da escola de música.
    Quando se trata de um pedido http://localhost:3001/instrumentos, o servidor responde com uma lista dos intrumentos da escola de música.
    Quando se trata de um pedido http://localhost:3001/, o servidor responde com um índice sendo que cada item do índice é um link para o http://localhost:3001/alunos, um link para http://localhost:3001/instrumentos e um link para http://localhost:3001/cursos.
