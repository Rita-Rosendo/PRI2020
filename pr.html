    • Abstract:

    Foi-nos proposta a realização de um trabalho de casa que tem como objectivo criar um servidor em NodeJs que recebe pedidos e responde com a página HTML correspondente.
    Primeiramente, foi necessário criar uma stylesheet que gera um índice multinível em que cada nível é um concelho e, em cada concelho, encontram-se os arqueossítios que pertencem ao mesmo. Esta stylesheet gera, dentro de uma pasta denominada "arqweb", os ficheiros html para cada arqelem e o index.html.
    De seguida, foi necessário criar um servidor que ao receber o pedido localhost:7777/arqs/ 3 devolve o arq3.html, por exemplo, e que ao receber o pedido localhost:7777/arqs/* devolve o index.html.
    Depois de proceder à realização das tarefas ditas anteriormente, pode-se observar o seguinte: o servidor, depois de receber um pedido do tipo localhost:7777/arqs/ n em que n corresponde ao ficheiro do arqelem n, devolve o arqn.html; o servidor, depois de receber um pedido do tipo localhost:7777/arqs/*, devolve o index.html e este índice é navegável, ou seja, podemos, ao escolher um dos arqelem, ir para a página html desse arqelem porém, para voltar ao índice, é necessário efetuar novamente o pedido localhost:7777/arqs/*.
